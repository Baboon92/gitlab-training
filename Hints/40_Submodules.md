# Git Submodules

* Pointer to a specific commit in remote repository
* Independant sub directory


!SLIDE smbullets

~~~SECTION:handouts~~~

****

Syntax is <git commit> <directory> (<remote origin>):

    $ git submodule
     ad5e30952f3971eea7a52936ccfed1fd904c9fa4 global (heads/master)
